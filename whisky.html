Stap 1: Whisky Installeren

Optie A: Via Homebrew (Aanbevolen)

# Installeer Homebrew als je die nog niet hebt /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" # Voeg deze regel toe aan je ~/.zprofile als Homebrew dat aangeeft echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile eval "$(/opt/homebrew/bin/brew shellenv)" # Installeer Whisky brew install --cask whisky

Optie B: Handmatige Download

# Download de nieuwste Whisky release cd ~/Downloads curl -L -o Whisky.zip https://github.com/Whisky-App/Whisky/releases/latest/download/Whisky.zip # Pak uit en verplaats naar Applications unzip Whisky.zip mv Whisky.app /Applications/
Tip: Homebrew is de aanbevolen methode omdat het updates automatisch beheert.

Stap 2: Whisky Opstarten en Configureren

# Open Whisky (na installatie) open -a Whisky

Eerste setup in Whisky:

  1. Whisky zal vragen om GPTK (Game Porting Toolkit) te installeren - dit is ESSENTIEEL
  2. Klik "Install" en volg de prompts
  3. Dit kan even duren (ongeveer 1-2 GB download)
Belangrijk: Zorg dat je een stabiele internetverbinding hebt voor de GPTK download.

Stap 3: Een Nieuwe Bottle Maken

In de Whisky GUI:

  1. Klik op "Create New Bottle"
  2. Geef het een naam (bijv. "GeometryDash")
  3. Kies als Windows versie: Windows 10
  4. Kies als bottle type: Gaming (aanbevolen voor games)
Tip: De "Gaming" bottle is geoptimaliseerd voor betere game prestaties.

Stap 4: Je .exe Bestand Installeren

Methode 1: Slepen en Droppen

  • Sleep je Geometry Dash .exe bestand direct naar het Whisky venster
  • Whisky start de installer automatisch op

Methode 2: Via het Bottle Menu

  1. Klik op je zojuist gemaakte bottle
  2. Klik op "Run" om de bottle te openen
  3. In het bottle venster: File > Install .exe
  4. Selecteer je Geometry Dash .exe bestand

Stap 5: Geometry Dash Specifieke Instellingen

Voor Geometry Dash kun je deze extra stappen proberen voor betere prestaties:

DXVK Instellen (voor betere graphics):

# Whisky heeft dit meestal al geconfigureerd, maar je kunt controleren: # In Whisky -> Selecteer je bottle -> Settings -> DXVK Backend # Zet dit aan als het niet standaard aan staat

Audio Instellingen:

  • In bottle settings: Audio > Driver = Apple Audio

Prestatie Tips:

  • Zet graphics op "Low" in Geometry Dash zelf
  • Sluit andere apps tijdens het spelen
  • Gebruik een externe muis voor betere respons
  • Toetsenbord controls werken meestal beter dan touchpad

Stap 6: Het Spel Uitvoeren

Na installatie:

  1. In het bottle venster zie je Geometry Dash in de programma lijst
  2. Dubbelklik op GeometryDash.exe om te starten

Tip: Als het spel niet goed werkt, probeer dan een andere Windows versie (Windows 7) in de bottle settings.

Veelvoorkomende Problemen

  • Als het crasht: Probeer een andere Windows versie (Windows 7 i.p.v. 10)
  • Als audio hapert: Zet DXVK uit in bottle settings
  • Als prestaties slecht zijn: Zet graphics kwaliteit lager in het spel
Waarschuwing: Geometry Dash kan via Wine niet perfect werken. Verwacht mogelijke audio-glitches of prestatieproblemen.
